# ✓ Chore Checklist

A beautiful, modern chore tracking application built with vanilla HTML, CSS, and JavaScript. Featuring a responsive design optimized for mobile devices with smooth animations and a glassmorphism UI.

![Chore Checklist Preview](https://img.shields.io/badge/Status-Live-green) ![Version](https://img.shields.io/badge/Version-1.0.0-blue) ![License](https://img.shields.io/badge/License-GPL--3.0-red)

## ✨ Features

- 📱 **Mobile-First Design** - Optimized for iPhone and all devices
- 🎨 **Modern UI** - Glassmorphism design with smooth animations
- 📊 **Progress Tracking** - Real-time completion statistics with progress bar
- 🌙 **Dark Mode** - Automatic system theme detection
- ⚡ **Fast & Lightweight** - No frameworks, pure vanilla JavaScript
- 🔧 **Keyboard Shortcuts** - Cmd/Ctrl + Enter to create checklist, Esc to clear
- ♿ **Accessible** - Full screen reader support and WCAG compliance
- 🎯 **PWA Ready** - Progressive Web App capabilities
- 🚀 **Zero Build** - No compilation needed, works instantly
